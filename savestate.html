01. OVERVIEW
Save State is a puzzle game where the player manipulates objects to solve mechanical problems. The objective is to build a video game command by correctly placing pieces and respecting the logic of assembly. My main responsibility was implementing the Assembly System and Progression System to ensure a cohesive and intuitive player experience.
02. TECHNICAL CHALLENGE: Assembly System
Created a system where the player can build a command by placing multiple pieces, respecting a logical order and physical structure. I used Unreal Engine's Socket system in conjunction with the AttachToComponent method to ensure each piece fits precisely and realistically into the defined points of the main command model.
03. TECHNICAL CHALLENGE: Progression System
Allow the player to build the command in their own way and deliver it when they want, but ensure the game can automatically evaluate if the assembly was done correctly or not. This was an interesting challenge. I implemented a validation system at delivery time that checks if all required pieces are correctly attached to their respective sockets.
